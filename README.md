# Deep Learning Practice

Short notebooks and small datasets that cover core deep learning concepts, from basic ANN/CNN comparisons to text embeddings and training pipelines. Each folder focuses on one topic with a minimal, hands-on example.

## Contents
- CNN vs ANN
- Image classification
- Handwritten digits classification
- Customer churn prediction (ANN)
- Data augmentation
- Dropout regularization
- Gensim Word2Vec
- TensorFlow pipeline
- Word embedding

## Notes
- Files are organized by topic in folders and notebooks.
- Datasets are included where needed for the exercises.
- Notebooks are meant for practice and quick reference rather than full projects.
